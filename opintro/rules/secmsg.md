# 信息安全防护

信息安全防护功能，用于针对用户源站对WAF请求回源的响应码，所做的二次处理操作，根据用户自行设置的规则，可进行响应内容阻断或者返回用户自定义的页面。

![](/images/opintro/waf56.png)

添加规则参数说明

  - 规则名称：规则的名称
  - 防护类型：信息过滤 和 信息伪装
  - 过滤类型：指定的响应码（400-700）

![](/images/opintro/waf57.png)

说明： 信息过滤指的是触发规则的响应内容会阻断掉，并返回断开响应的信息给请求端。
信息伪装用户可自行定义返回的文件内容，支持TXT和HTML格式。
